```typescript
export type StrategyId = 'speech' | 'r1-f0' | 'r1-2f0' | 'r1-3f0' | 'r2-2f0' | 'r2-3f0' | 'singer-formant';
export type StrategyMode = 'off' | 'overlay' | 'locked';

export interface StrategyDefinition {
  id: StrategyId;
  notation: string;       // e.g., "R1:f0"
  description: string;    // e.g., "First formant tracks pitch"
  controls: ('f1' | 'f2' | 'f3' | 'f4' | 'f5')[];
  f0Range: { min: number; max: number };
  voiceTypes: string[];
}

export interface StrategyResult {
  targets: {
    f1: number | null;
    f2: number | null;
    f3: number | null;
    f4: number | null;
    f5: number | null;
  };
  inRange: boolean;
  clamped: boolean;
}
```

6. Create `src/lib/strategies/engine.ts`:
Export `computeTargets(strategyId: StrategyId, f0: number, voiceType: string): StrategyResult`.

Formant range boundaries for clamping:
- F1: min 200, max 1000
- F2: min 600, max 3000
- F3: min 1500, max 3500
- F4: min 2500, max 5000
- F5: min 3500, max 6000

Strategy computations:
- `speech`: all targets null, inRange=true, clamped=false
- `r1-f0`: f1 = f0, clamp to [200, 1000]
- `r1-2f0`: f1 = 2*f0, clamp to [200, 1000]
- `r1-3f0`: f1 = 3*f0, clamp to [200, 1000]
- `r2-2f0`: f2 = 2*f0, clamp to [600, 3000]
- `r2-3f0`: f2 = 3*f0, clamp to [600, 3000]
- `singer-formant`: look up center by voice type from table:
  bass=2384, baritone=2454, tenor=2705, alto=2800, mezzo=2900, soprano=3092, child=3200 (default 2600).
  f3 = center - 200, f4 = center, f5 = center + 300.
  Clamp each to its range. inRange = f0 < 660.

inRange check: compare f0 against the strategy's f0Range from STRATEGY_DEFINITIONS.
clamped: true if the computed target was outside its formant range boundary.
