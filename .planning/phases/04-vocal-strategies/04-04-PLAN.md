  <how-to-verify>
Run `npm run dev` and open the app in Chrome.

**1. STRAT-05: Strategy selector (D-01, D-02, D-03, D-05, D-06)**
- Verify the strategy panel is visible below the vowel chart
- Verify "Speech (untuned)" is the default active strategy
- Verify each strategy shows notation (e.g., "R1:f0") and a plain-language description
- Verify at least 7 strategy presets are listed
- Click "Auto" button and verify it selects a strategy appropriate for the current voice type

**2. STRAT-02: Overlay mode (D-07, D-08, D-09, D-10)**
- Select "R1:2f0" strategy
- Set mode to "Overlay"
- Click Start audio and play a note
- Verify: amber/orange target lines appear on the piano view (D-07)
- Verify: amber target marker (open circle) appears on the F1/F2 chart with a connecting line from the current position (D-08)
- Change f0 (via QWERTY keys or pitch slider) and verify target lines/markers move in real time (D-09)
- Verify: audio does NOT change when overlay is on (overlay is visual-only)
- Verify: all overlay elements use the same color (D-10)

**3. STRAT-03: Locked mode (D-11, D-12)**
- With "R1:2f0" selected, switch mode to "Locked"
- Change f0 by playing different notes
- Verify: F1 audibly changes to track 2*f0 (the vowel sound shifts)
- Verify: F1 changes smoothly, no clicks or zipper noise (D-11)
- Verify: the drag handle on the F1/F2 chart moves automatically
- Push f0 high enough that 2*f0 > 1000 Hz (F1 max)
- Verify: overlay/target turns red or shows warning indicator (D-12)
- Verify: formant stops at 1000 Hz, does not go beyond

**4. STRAT-04: Drag override (D-14, D-15)**
- With "R1:2f0" locked, drag the vowel handle on the F1/F2 chart
- Verify: the connecting line becomes dashed during drag (D-15)
- Verify: you can move the formant freely while dragging
- Release the drag handle
- Verify: the formant snaps back to the strategy target position

**5. Singer's formant cluster (D-13)**
- Select "Singer's formant" strategy, mode "Locked"
- Select a bass or baritone voice preset
- Verify: F3, F4, and F5 cluster near the center frequency for that voice type
- Verify: you can hear a perceptible spectral peak (the "ring") in the sound
- Switch to soprano voice preset
- Verify: the cluster center shifts higher

**6. Overall**
- Switch between different strategies and verify mode toggle works for each
- Verify no audio glitches when switching strategies
- Verify all existing functionality (vowel presets, voice presets, phonation, expression) still works
