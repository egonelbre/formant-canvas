1. **Reactive decomposition** using `$derived`:
   ```typescript
   let decomp = $derived(rdToDecomposition(rd, f0));
   let params = $derived(computeLfParams(rd, f0));
   ```

2. **Annotated SVG waveform** — renders the LF derivative waveform with timing markers:
   - SVG viewBox sized to container width (bind:clientWidth) x 120px height
   - Margins: left=40 (for y-axis space), right=10, top=10, bottom=25 (for x-axis labels)
   - Generate 300 sample points over one period T0 using `lfDerivativeSample`
   - Normalize amplitude to fit plot area
   - Draw the waveform as an SVG `<path>` with stroke `var(--color-text)`, stroke-width 1.5

3. **Timing markers** (vertical dashed lines at key instants on the waveform):
   - **Tp** (max flow): vertical dashed line at x = Tp/T0 * plotWidth, color `var(--color-accent)`, label "Tp" below
   - **Te** (closure): vertical dashed line at x = Te/T0 * plotWidth, color `var(--color-warning, #e67e22)`, label "Te" below
   - **Ta** (return phase): horizontal bracket or arrow from Te to Te+Ta, label "Ta" (show the return phase duration)
   - **Tc** (complete closure): if Tc < T0, vertical dashed line; typically Tc = T0, so just label the end "T0"

4. **Numeric readouts** — a row of labeled values below the waveform SVG:
   ```
   Ra: 0.038  |  Rk: 0.342  |  Rg: 1.10  |  Ta: 0.32ms
   ```
   Use a CSS grid or flex row with monospace font for alignment. Values update reactively via `$derived`. Format:
   - Ra: 3 decimal places
   - Rk: 3 decimal places
   - Rg: 2 decimal places
   - Ta: display in milliseconds with 2 decimal places (Ta * 1000)

5. **Styling**: match existing component style patterns:
   - Use `var(--color-text)`, `var(--color-text-secondary)`, `var(--color-border)`, `var(--color-accent)` CSS variables
   - Font sizes: 10px for marker labels, 11px for readout values, 9px for readout labels
   - Compact layout — the whole panel should be ~160px tall (120px SVG + 40px readouts)
