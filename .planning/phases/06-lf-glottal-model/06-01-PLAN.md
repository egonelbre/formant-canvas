Create `src/lib/audio/dsp/lf-model.ts` with these exports:

1. **`RdDecomposition` interface**: `{ Ra, Rk, Rg, Tp, Te, Ta, Tc }` (all numbers, seconds for timing params)

2. **`LfParams` interface**: `{ Tp, Te, Ta, T0, alpha, epsilon, E0, Ee }` (all numbers)

3. **`rdToDecomposition(Rd: number, f0: number): RdDecomposition`**:
   - Clamp Rd to [0.3, 2.7]
   - T0 = 1/f0
   - Ra = (-1 + 4.8 * Rd) / 100
   - Rk = (22.4 + 11.8 * Rd) / 100
   - Rg = (0.44 * Rd + 1.073) / (1.0 + 0.46 * Rd) (simplified Fant 1995 approximation)
   - Tp = T0 / (2 * Rg)
   - Te = Tp * (1 + Rk)
   - Ta = Ra * T0
   - Tc = T0
   - Return all values

4. **`solveEpsilon(Ta: number, Tb: number): number`** (Tb = T0 - Te):
   - Newton-Raphson solving: epsilon * Ta = 1 - exp(-epsilon * Tb)
   - Initial guess: 1 / Ta
   - Max 20 iterations, tolerance 1e-10
   - Fallback to bisection [0, 10/Ta] if Newton diverges

5. **`solveAlpha(Tp: number, Te: number, epsilon: number, Ta: number, T0: number): number`**:
   - Newton-Raphson solving the zero-net-flow constraint
   - The integral of the open phase must equal the negative integral of the return phase
   - Initial guess: 1/Tp
   - Max 20 iterations, tolerance 1e-10
   - Fallback to bisection if divergence

6. **`computeLfParams(Rd: number, f0: number): LfParams`**:
   - Call rdToDecomposition to get timing
   - Solve epsilon from Ta, Tb
   - Solve alpha from Tp, Te, epsilon, Ta, T0
   - Compute E0 from alpha, Tp (continuity at Te)
   - Ee = 1.0 (normalized)
   - Return LfParams

7. **`lfDerivativeSample(t: number, params: LfParams): number`**:
   - Open phase (0 <= t < Te): E0 * exp(alpha * t) * sin(PI * t / Tp)
   - Return phase (Te <= t < T0): (-Ee / (epsilon * Ta)) * (exp(-epsilon * (t - Te)) - exp(-epsilon * (T0 - Te)))
   - Closed phase: 0

Create `src/lib/audio/dsp/lf-wavetable.ts` with these exports:

1. **`fft(real: Float32Array, imag: Float32Array): void`** — in-place radix-2 Cooley-Tukey FFT. Input arrays must be power-of-2 length. This function is designed to be inlined into the AudioWorklet later.

2. **`ifft(real: Float32Array, imag: Float32Array): void`** — inverse FFT via conjugate-FFT-conjugate trick, with 1/N normalization.

3. **`generateLfWavetable(Rd: number, tableSize: number, maxHarmonics: number, sampleRate?: number): Float32Array`**:
   - Default sampleRate = 48000
   - Compute LF params via computeLfParams(Rd, 1/tableSize * sampleRate) — use a reference f0 so one period fills the table
   - Actually simpler: generate one analytical LF period of `tableSize` samples at a normalized f0 where T0 = tableSize samples
   - FFT the analytical period
   - Zero out bins above maxHarmonics (both positive and negative frequency bins)
   - IFFT to get band-limited version
   - Normalize peak to 1.0
   - Return Float32Array

4. **`OCTAVE_F0_BOUNDARIES: number[]`** — constant array of 10 f0 boundaries: `[55, 82.5, 110, 165, 220, 330, 440, 660, 880, 1320, 1760]` (10 bands = 11 boundaries, roughly half-octave to octave spacing to cover voice range)

5. **`generateLfWavetableSet(Rd: number, sampleRate: number): Float32Array[]`**:
   - For each of the 10 bands, compute maxHarmonics = floor(sampleRate / 2 / bandMaxF0)
   - Call generateLfWavetable(Rd, 2048, maxHarmonics) for each band
   - Return array of 10 Float32Arrays

6. **`RD_GRID: number[]`** — constant array of ~10 Rd values from 0.3 to 2.7 in steps of ~0.267: `[0.3, 0.57, 0.83, 1.1, 1.37, 1.63, 1.9, 2.17, 2.43, 2.7]`

7. **`generateFullWavetableBank(sampleRate: number): Float32Array[][]`**:
   - For each Rd in RD_GRID, call generateLfWavetableSet(Rd, sampleRate)
   - Return 2D array: [rdIndex][octaveIndex] = Float32Array
   - Total: 10 Rd values * 10 octave bands = 100 tables * 2048 * 4 bytes = ~800KB
