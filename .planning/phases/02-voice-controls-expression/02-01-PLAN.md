    vibrato.ts — `vibratoModulation(vibratoPhase: number, extentCents: number): number`
    - Returns a frequency ratio to multiply with f0
    - At vibratoPhase=0, returns 1.0 (no modulation)
    - At vibratoPhase=0.25 (peak of sine), returns 2^(extentCents/1200)
    - At vibratoPhase=0.75 (trough of sine), returns 2^(-extentCents/1200)
    - With extentCents=0, always returns 1.0
    - With extentCents=100 (one semitone), peak ratio is approximately 1.05946
    - Formula: 2^(extentCents * sin(2*PI*vibratoPhase) / 1200)

    Also export `advanceVibratoPhase(currentPhase: number, rate: number, sampleRate: number): number`
    - Advances phase by rate/sampleRate per sample
    - Wraps phase at 1.0

    jitter.ts — `computeJitterOffset(f0: number, jitterAmount: number): number`
    - Returns a Hz offset to add to f0
    - With jitterAmount=0, always returns 0
    - With jitterAmount=1, returns a value in range [-0.03*f0, +0.03*f0] (3% max deviation)
    - Result is random (uses Math.random internally)
    - At f0=100, jitterAmount=1, abs(result) <= 3
    - At f0=200, jitterAmount=0.5, abs(result) <= 3
  </behavior>

  <behavior>
    spectral-tilt.ts — One-pole low-pass filter per Klatt LpFilter1 pattern:
    - `computeTiltCoefficients(tiltDb: number, sampleRate: number): { a: number; b: number }`
      - tiltDb=0 returns a=1, b=0 (passthrough)
      - tiltDb=6 returns coefficients that attenuate 3kHz by ~6dB
      - Reference frequency is 3000 Hz (Klatt standard)
      - Formula: g = 10^(-tiltDb/20), w = 2*PI*3000/sampleRate, q = (1 - g*g*cos(w))/(1 - g*g), b = q - sqrt(q*q - 1), a = 1 - b
    - `applyTiltSample(input: number, prevOutput: number, a: number, b: number): number`
      - Returns a * input + b * prevOutput
      - With a=1, b=0: output equals input (passthrough)

    pitch-utils.ts:
    - `hzToNote(hz: number): { name: string; octave: number; cents: number }`
      - 440 Hz returns { name: 'A', octave: 4, cents: 0 }
      - 261.63 Hz returns { name: 'C', octave: 4, cents: ~0 }
      - 466.16 Hz returns { name: 'A#', octave: 4, cents: ~0 }
      - 450 Hz returns { name: 'A', octave: 4, cents: ~+39 }
    - `midiToHz(midi: number): number`
      - midi 69 returns 440
      - midi 60 returns ~261.63
    - `sliderToHz(position: number): number`
      - position 0 returns 55
      - position 1 returns 1100
      - position 0.5 returns geometric mean ~245.97 (sqrt(55*1100))
    - `hzToSlider(hz: number): number`
      - 55 returns 0
      - 1100 returns 1
      - Inverse of sliderToHz
    - `formatPitchReadout(hz: number): string`
      - 220 returns "220 Hz . A3 . +0c" (per D-05)
      - 450 returns "450 Hz . A4 . +39c"
  </behavior>
