**HarmonicBars.svelte:**

Props:
```typescript
interface Props {
  freqToX: (freq: number) => number;
  barRegionHeight: number;  // 72 (80px region minus 8px top margin)
}
```

Reactive computation using $derived.by():
- Compute harmonics: for n=1 to 24, freq = voiceParams.f0 * n. Break when freq > 1976 (B6).
- For each harmonic, compute amplitude via spectralEnvelope(freq, voiceParams.formants)
- Find maxAmplitude across all harmonics for normalization
- For each harmonic:
  - x = freqToX(freq)
  - barHeight = (amplitude / maxAmplitude) * barRegionHeight
  - y = 80 - barHeight (bars grow upward from key top at y=80, with 8px margin from SVG top)

Rendering:
- Each bar: `<rect>` width=4, centered on x (x - 2), fill depends on harmonic number:
  - n === 1 (fundamental): #6366f1 (accent), opacity 1.0
  - n > 1 (overtones): #e0e0e0, opacity 0.8
- At least 12 harmonics rendered (D-06) -- the loop goes to 24 but breaks at B6 limit
- Render as SVG `<g>` group (no standalone SVG)

**FormantCurves.svelte:**

Props:
```typescript
interface Props {
  freqToX: (freq: number) => number;
  curveRegionHeight: number;  // 72 (same as bar region)
}
```

Formant colors (per UI-SPEC):
```typescript
const FORMANT_COLORS = ['#f97316', '#22c55e', '#3b82f6', '#a855f7']; // F1-F4
const FORMANT_LABELS = ['F1', 'F2', 'F3', 'F4'];
```

**Response curves (D-07):**
- For each of the 4 formants, compute ~200 sample points across C2-B6 range (65-1976 Hz)
- At each sample frequency: x = freqToX(freq), amplitude = formantMagnitude(freq, formant)
- Normalize amplitude: find global max across all 4 formants at all sample points, then y = 80 - (amplitude / maxAmplitude) * curveRegionHeight
- Build SVG path string: M x0,y0 L x1,y1 L x2,y2 ...
- Render each curve as `<path d={pathString} stroke={color} stroke-width="2" fill="none" opacity="0.7" />`
- Use $derived.by() so curves only recompute when formant params change (not on every f0 change)

**Formant center markers (PIANO-04):**
- For each formant (F1-F4): if freq is within C2-B6 range (65-1976 Hz):
  - x = freqToX(formant.freq)
  - Draw vertical dashed line: `<line>` from y=8 to y=80, stroke={color}, stroke-width=2, stroke-dasharray="2 2", opacity=0.5
  - Label at top: `<text>` "F1"/"F2"/"F3"/"F4" at (x, 6), 11px semibold, fill={color}

Render as SVG `<g>` group.
